## 1. Purpose

Defines how a lead is classified into one of the fixed Tenacious ICP segments.

## 2. Inputs

* hiring signal brief
* AI maturity score
* company profile
* contradictory signal notes if any

## 3. Steps

1. load classification context
2. assess segment fit
3. identify alternate segment if relevant
4. assign confidence
5. abstain if below threshold
6. persist classification result
7. update lead state and CRM
