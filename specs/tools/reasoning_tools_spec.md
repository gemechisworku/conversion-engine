## 1. Purpose

Defines structured reasoning-support tools that convert evidence into constrained outputs.

## 2. Tool Set

* `score_ai_maturity`
* `classify_icp`
* `bench_match`
* `validate_claims`
* `compute_signal_confidence`

## 3. Rules

* outputs must be structured
* tools should preserve uncertainty explicitly
* no side effects beyond logging and optional result persistence
* these tools are decision-support, not final policy bypasses
